# Microsoft 365 IT Operations Dashboard

## Overview

This project presents an interactive Microsoft 365 IT Operations Dashboard developed to monitor operational KPIs, helpdesk performance, ticket resolution trends, infrastructure usage, and business intelligence reporting.

The dashboard provides insights into enterprise IT operations and supports data-driven decision-making through interactive Tableau visualizations.

---

## Project Objectives

- Monitor IT operational performance
- Analyze helpdesk ticket trends
- Track ticket resolution efficiency
- Evaluate system downtime patterns
- Improve operational visibility
- Support business intelligence reporting
